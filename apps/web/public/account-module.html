1Workspace settings — /settings

A workspace is the container for everything you do in AdGenieHQ: its members, connected accounts, campaigns, agent runs, recommendations, reports, plan, and billing all belong to one workspace and never leak across to another.

  • Name & identity — rename the workspace any time (Admin+). Each workspace has its own slug.
  • Multiple workspaces — you can belong to several (e.g. one per brand or client) and switch between them with the workspace picker. Your role can differ in each.
  • Publish webhook (Admin+) — an optional CMS hook for pushing content out; the secret is encrypted at rest with the same key as OAuth tokens and never returned to the browser.
  • Isolation — data is scoped per workspace at the API layer, so members of one workspace can never see another's records.

Create a second workspace

Use the workspace switcher → New workspace. Whoever creates it becomes its Owner. Agencies typically run one workspace per client so limits, integrations, and billing stay cleanly separated.

2Team & roles — /settings · Team

Invite teammates by email (Admin+). Invitations expire and can be revoked before they're accepted; the new member joins with the role you pick (default Marketer). Every workspace keeps at least one Owner — you can't demote the last one.

  1. Invite — enter an email and choose a role. You can never grant a role higher than your own.
  2. Accept — the invitee opens the link and joins the workspace.
  3. Manage — change a member's role or disable them. A non-Owner can't modify a peer at or above their own rank.

Roles & permissions matrix

Five roles, ranked Owner > Admin > Marketer > Analyst > Viewer. Higher roles inherit everything below them. This is exactly what the backend enforces:

3Integrations — /settings · Integrations

This is the only place OAuth "connections" live. Connecting and disconnecting is an Admin+ action; syncing data is Marketer+. Access tokens are Fernet-encrypted at rest and never exposed to the browser — all provider calls run server-side.

4Bring-your-own AI keys — /settings · AI keys

Prefer to run agents on your own LLM account? Add a provider credential (Admin+) and AdGenieHQ will use it for AI work in this workspace.

  • Encrypted, one active per provider — the secret is Fernet-encrypted at rest; only a cosmetic last-four hint is shown. Adding a new key for a provider automatically revokes the previous one.
  • Test button — validates the key against the provider and records the result (OK / failed) so you can see "last tested — OK" in the UI.
  • Optional — agents work without a key thanks to deterministic fallbacks; your own key makes outputs richer.

✅ Your key, your account

BYOK keys are never shown back to the browser after saving and can be revoked (Admin+) at any time. Plaintext is stored nowhere — only the encrypted ciphertext.

5API keys — /settings · API keys

Programmatic access to a workspace uses scoped API keys. Creating and revoking keys is Owner-only.

  1. Create — give it a label and the role it should grant when used (default Marketer); optionally set an expiry.
  2. Copy once — the full key is shown exactly once at creation in the form ak_<prefix>.<secret>. Store it now; the secret is SHA-256 hashed and unrecoverable.
  3. Use it — calls act with the key's role, scoped strictly to this workspace. last_used_at tracks activity.
  4. Revoke — kill a key instantly; revoked keys stop working immediately.

Keys inherit a role

An API key can never do more than the role it was minted with — a Marketer key can't approve a high-risk change or manage billing. Give each integration the least privilege it needs.

Prices are display-only; the actual charge lives on the matching PayPal Billing Plan. Starter/Pro also cap outbound writes (1,000 / 5,000 per month); Agency is unlimited.

Subscribe with PayPal

  1. Choose a plan & interval (monthly or annual) on the Billing page.
  2. Approve at PayPal — AdGenieHQ creates the subscription server-side and redirects you to PayPal's approval page.
  3. Activates on webhook — once you approve, PayPal fires BILLING.SUBSCRIPTION.ACTIVATED (signature-verified) and your plan goes live.
  4. Manage or cancel — Manage plan opens your PayPal autopay list, where you can view and cancel the subscription.

What counts against your plan

  • AI credits — a single monthly pool (rolling 30 days). Each AI action deducts credits: agent run 10, image generation 10, content draft 5, A/B test 3, outreach email 2. Run out and AI actions are blocked with a clear "upgrade for more credits" message — nothing fails silently. Agency plans never block.
  • Non-AI caps — landing pages, team seats, and outbound provider writes are enforced separately from credits.

Platform fees

Separate from your subscription, AdGenieHQ can charge platform fees on campaigns (a listing fee plus run fees — flat and/or a percentage of spend). You can preview them with a fee quote before launching a campaign, and see accrued fees under Billing → Fees and Invoices. Platform-fee invoices are billed via PayPal invoice.

?FAQ & tips

Who can connect an ad account?

An Admin or Owner. Marketers can sync and act on already-connected accounts, but connecting/disconnecting is Admin+ and always audit-logged.

Are my tokens ever exposed to the browser?

No. OAuth tokens, BYOK AI keys, and webhook secrets are encrypted at rest and never sent to the frontend — every provider call happens server-side.

How do I cancel my subscription?

Billing → Manage plan, which sends you to your PayPal autopay list. Cancel it there; PayPal notifies AdGenieHQ via webhook and the plan winds down.

What are platform fees, and how are they different from my plan?

Your plan is a recurring subscription (PayPal). Platform fees are per-campaign charges (listing + run fees) billed separately as a PayPal invoice. Preview them with a fee quote before you launch.

Do I have to bring my own AI key?

No. Agents have deterministic fallbacks, so they produce real output with no key configured. Add your own OpenAI / Anthropic / Google AI key only if you want richer results on your own account.

What's the safest way to onboard a client or stakeholder?

Invite them as an Analyst or Viewer — full read access to dashboards and reports, zero ability to run agents, change campaigns, spend, or touch billing.
